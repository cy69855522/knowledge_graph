岗位职责

1. 负责大模型训练所需的底层基础设施建设与维护，包括 GPU 集群、存储系统、任务调度与资源管理等；
2. 与训练工程、模型研究团队合作，优化分布式训练作业的部署流程，提高集群资源利用率与稳定性；
3. 搭建训练任务的监控与告警体系，对 GPU 利用率、网络带宽、IO 性能等关键指标进行观测和调优；
4. 参与建设内部 AI 平台工具（如一键提任务、实验管理、日志系统等），提升研发团队整体效率；
5. 编写并维护相关技术文档和运维手册，持续改进系统的可靠性与可维护性。

岗位要求

1. 计算机、软件工程、网络工程等相关专业硕士及以上学历；
2. 熟悉 Linux 操作系统，精通至少一种后端编程语言（如 Python / Go / C++），有分布式系统开发经验；
3. 熟悉主流容器与编排技术（Docker、Kubernetes 等），了解常见任务调度与资源管理框架（如 Slurm、K8s Operator 等）者优先；
4. 对GPU计算、分布式训练、网络与存储有基本理解，有大规模训练平台或云原生平台建设经验者优先；
5. 具备良好的问题定位与性能调优能力，能在复杂系统中快速排查并解决问题；
6. 具备良好的沟通与协作能力，能够支持多团队协同工作。
