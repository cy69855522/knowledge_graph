# 离职的时候聊一些什么东西？

- 写bug会引起非常大的风险，怎么降低风险是AD公司的核心。
- 成本太高，讲PPT需要做很多准备，初心是为了让大家紧跟前沿。效率是公司的护城河，精力有限，如何保证快速吸收消化？开议疲劳。

### 我感觉在公司工作的时候经常被会议或者issue中断，这种心流破坏会导致项目效率大大降低？从生理层面如何解释这种现象？


这种被迫中断心流（Flow State）的情况在工程开发中确实非常致命。从生理和认知神经科学的角度来看，这不仅仅是“心情烦躁”，而是大脑在高负荷运转时遭遇了物理意义上的“系统中断”。

以下是该现象的生理学解释：

#### 1. 认知切换成本与“注意力残留”

当你进入“心流”时，大脑的前额叶皮层（Prefrontal Cortex）正在协调极其复杂的逻辑和上下文。

* **注意力残留（Attention Residue）：** 即使你离开了会议回到工位，你的大脑并不能像开关一样瞬间切换。苏菲·勒罗伊（Sophie Leroy）教授的研究发现，当你从任务 A 切换到任务 B 时，一部分注意力仍会“滞留”在任务 A 上。
* **生理代价：** 这种残留会导致认知容量下降，你会发现自己盯着屏幕却半天进入不了状态，这就是大脑在清理“背景进程”的物理延迟。

#### 2. 执行功能的“冷启动”瓶颈

大脑处理复杂工作依靠的是 **执行功能（Executive Function）** 。

* **工作记忆的清空：** 编写复杂代码或分析架构时，你的工作记忆（Working Memory）里装满了临时的变量、逻辑链条和指针。
* **物理中断 = 数据丢失：** 突如其来的会议或 Issue 会瞬间清空这些“高速缓存”。当你重新开始时，大脑必须重新加载这些数据。这种“冷启动”过程会大量消耗 ATP（细胞能量货币），导致你感到生理上的疲惫。

##### 3. 皮质醇与去甲肾上腺素的干扰

频繁的中断会触发身体的微型“压力响应”。

* **应激反应：** 当你在深度思考时被叫醒或弹窗干扰，大脑的杏仁核会感知到一种“威胁”，导致**皮质醇（Cortisol）**和**去甲肾上腺素**小幅上升。
* **心流抑制：** 心流的产生通常伴随着多巴胺和内啡肽的分泌，而高水平的皮质醇会拮抗这些神经递质，直接从生化层面切断心流环境。

---

#### 如何在生理层面优化？

既然大脑存在物理限制，我们可以通过以下策略减少损害：

* **建立“外部存储”减少加载负担：** 既然用纸张记录容易丢， **强烈建议使用支持全平台同步的数字记事本（如 Notion, Obsidian 或手机自带备忘录）** 。在去开会前的 30 秒，迅速敲下你当前脑子里的“逻辑断点”和“下一步操作”。这相当于给大脑做一个“快照”，下次回来时能极大地降低重载（Reload）的生理功耗。
* **碎片时间“批量处理”：** 尝试将所有的 Issue 回复和非紧急沟通集中在某个固定时段（如午饭前或下班前），给大脑预留出至少 90-120 分钟的“无干扰生理期”。
* **利用物理环境暗示：** 佩戴降噪耳机或改变工位光线，这些感官信号可以帮助大脑更快分泌进入心流所需的神经递质。
